Vagrant Setup

1) Install Virtual Box (https://www.virtualbox.org/wiki/Downloads) and Vagrant (http://vagrantup.com/)

2) vagrant box add precise32 http://files.vagrantup.com/precise32.box

3) run "vagrant up"

4) access the VM using "vagrant ssh"
